# Käyttäjän poisto

- Vaatii kirjautumisen
- Käyttäjä voi poistaa vain omat tietonsa

**URL**: `{host}/users/{id}`

**Metodi**: `DELETE`


## Esimerkkipyyntö:

**request parameter**: `{id}`: Poistettavan käyttäjän id

**request header**: `Authorization: {token}`


## Esimerkkivastaus:

**response status:** `200 OK`
